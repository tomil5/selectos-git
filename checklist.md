**ПО:** Git 

**ОС:** SELECTOS (на базе Debian)

| # | Проверка | Авто |
|---|----------|------|
| 1 | `apt-get install git` выполняется без ошибок | — |
| 2 | `git --version` возвращает строку вида `git version X.Y.Z` | ✔ |
| 3 | Версия Git >= 2.x | ✔ |
| 4 | `git init` создаёт директорию `.git` | ✔ |

| 9 | `git merge` сливает ветки без конфликта | ✔ |
| 10 | Конфликт при merge корректно детектируется | ✔ |
| 11 | `git tag` создаёт и удаляет теги | ✔ |
| 12 | Git работает с файлами, у которых пробелы в имени | ✔ |
| 13 | `git clone` клонирует публичный репозиторий | — |
